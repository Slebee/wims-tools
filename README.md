# Wims-tools

wims前端项目工具，用于管理项目相关仓库

### 快速开始
```sh
$git clone https://github.com/Slebee/wims-tools.git
$npm i
$npm link
```
linux系统下注意权限

# 命令

### `wims create`
创建模块目录，创建的默认目录请看 [wims-module-template](https://github.com/Slebee/wims-module-template) master

### `wims add|a`

添加repository

### `wims list|l`

查看添加repository

### `wims init|i`

根据提示输入来下载repository

### `wims version`

查看工具版本
